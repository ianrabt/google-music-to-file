Output Format

Playing

Google Music is open in a tab and currently playing music

Paused

Google Music is open in a tab and paused

Stopped

Google Music is either not open or has nothing in the queue (note: variables are not available and characters do not need to be escaped)


Temporary File



Variable Truncation

Truncate variables that are over a certain length.






Syntax Help

Google Music to File offers a very simple way to format the text output in its temporary file.

  • $artist — displays the currently playing artist
  • $album — displays the currently playing album
  • $title — displays the currently playing song title
  • \$ — escapes the "$" character
  • \\ — escapes the "\" character
